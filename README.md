# The Hidden Ridge Studio — Website

### Overview
This repo hosts the official THRS site, showcasing our flagship venture **MMNTM Rugby Hub** — a Kenyan-born, culture-driven sports-tech platform.

### Tech Stack
- **Next.js 14** + **Tailwind CSS**
- **Shadcn/UI** component library
- **Supabase** for form & data
- **Vercel** for deployment
- **Google Analytics 4** for tracking

### Branching
- `main` → production
- `dev` → staging
- `feature/*` → individual updates

### Run Locally
```bash
npm install
npm run dev
